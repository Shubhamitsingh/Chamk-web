# Images Directory

## Portrait Image for Phone Mockup

Please add your portrait image here with the filename:
- `portrait-girl.jpg` (or `.png`, `.webp`)

### Image Requirements:
- Recommended size: 1080x1920px (9:16 aspect ratio for phone display)
- Format: JPG, PNG, or WebP
- Portrait of a girl/woman for the hero section phone mockup

### Image Path:
The image will be automatically loaded from: `/images/portrait-girl.jpg`

If the image is not found, a gradient placeholder will be shown instead.
