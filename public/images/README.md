# Images Directory

## App Logo

Please add your app logo here with the filename:
- `logo.png` (or `.svg`, `.jpg`, `.webp`)

### Logo Requirements:
- Recommended size: 200x200px or higher (square format works best)
- Format: PNG (with transparency), SVG, JPG, or WebP
- The logo will be displayed in the Header and Footer at 40x40px (h-10 w-10)

### Logo Path:
The logo will be automatically loaded from: `/images/logo.png`

**Note:** If you want to use a different filename, update the `src` attribute in:
- `src/components/Header.jsx` (line ~43)
- `src/components/Footer.jsx` (line ~43)

## Portrait Image for Phone Mockup

Please add your portrait image here with the filename:
- `portrait-girl.jpg` (or `.png`, `.webp`)

### Image Requirements:
- Recommended size: 1080x1920px (9:16 aspect ratio for phone display)
- Format: JPG, PNG, or WebP
- Portrait of a girl/woman for the hero section phone mockup

### Image Path:
The image will be automatically loaded from: `/images/portrait-girl.jpg`

If the image is not found, a gradient placeholder will be shown instead.
